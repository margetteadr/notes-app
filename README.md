# notes-app
## Project Overview
This project is a Java based desktop application created as part of a school assignment. 
The application allows users to log in and manage notes using a simple interface/window. 

## Features
- Login screen
- a created dashboard
- Create, update, and delete notes.
- simple and user-friendly design

## Requirements
- JAVA JDK installed
- VS Code

  ## How to run the code
  - Download the repositry.
  - Open the application in VS Code.
  - Make sure Java is installed.
  - Open the main Java file.
  - Run the program.
